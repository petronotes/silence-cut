2 Параметры обнаружения тишины
-40 dB

Уровень ниже которого аудио считается тишиной. -40 dB рекомендуется для большинства видео.

0.5 с

Минимальная продолжительность паузы, которая будет вырезана. Короче — и паузы в речи тоже удалятся.

0.10 с

Оставить небольшую паузу перед началом звука, чтобы монтаж звучал естественно.

0.10 с

Небольшой «хвост» после конца звука — предотвращает обрыв на полуслове.

0.5 с

Длинные паузы заменяются коротким промежутком вместо полного удаления — монтаж дышит.

0.05 с

Объединяет соседние тихие участки. Убирает «мерцание» при словах с паузами внутри.

Как это работает?
1Декодирование аудио — браузер декодирует звуковую дорожку через Web Audio API (AudioContext.decodeAudioData) без обращения к серверу.
2Анализ RMS — видео разбивается на маленькие окна (~20 мс), для каждого вычисляется среднеквадратическое значение (RMS) громкости в дБ.
3Маркировка участков — окна ниже порога помечаются как «тишина». Соседние тихие окна объединяются с учётом параметра «Сглаживание».
4Монтаж через MediaRecorder — исходный <video> ускоренно проигрывается, активные сегменты захватываются через canvas + MediaRecorder API. Тишина пропускается.
5Сборка WebM/MP4 — результирующие чанки собираются в Blob, который можно скачать напрямую из браузера.
Ограничения браузера

⚠️ Формат выхода — WebM (видео + аудио через MediaRecorder). Это ограничение браузерного API; перекодирование в MP4 без сервера невозможно.

⚠️ Большие файлы (>500 МБ) могут замедлять работу из-за ограничений оперативной памяти вкладки. Для профессиональных нужд рекомендуется использовать Python-скрипт ниже.

⚠️ Некоторые форматы кодеков (H.265/HEVC, некоторые MOV) могут не поддерживаться — зависит от браузера и ОС.

Python-скрипт (для продвинутых)

Для обработки больших файлов, сохранения оригинального кодека и полного контроля над форматом используйте скрипт на Python с moviepy и librosa:

# Установка зависимостей
pip install moviepy librosa soundfile numpy

# Запуск
python silence_cut.py input.mp4 output.mp4 --threshold -40 --min-silence 0.5 --pad 0.1
"""
silence_cut.py — Вырезатель тишины
Использование: python silence_cut.py input.mp4 output.mp4 [опции]
"""

import argparse, numpy as np, librosa
from moviepy.editor import VideoFileClip, concatenate_videoclips

def detect_silence(audio, sr, threshold_db, min_silence_sec, smooth_sec, pad_before, pad_after, max_keep):
    frame_len = int(sr * 0.02)
    hop_len = frame_len // 2
    rms = librosa.feature.rms(y=audio, frame_length=frame_len, hop_length=hop_len)[0]
    rms_db = 20 * np.log10(rms + 1e-9)
    times = librosa.frames_to_time(np.arange(len(rms_db)), sr=sr, hop_length=hop_len)
    is_silent = rms_db < threshold_db

    # Сглаживание
    smooth_frames = int(smooth_sec / (hop_len / sr))
    from scipy.ndimage import uniform_filter1d
    is_silent = uniform_filter1d(is_silent.astype(float), size=smooth_frames) > 0.5

    # Группировка
    segments, active, start = [], False, 0.0
    for i, silent in enumerate(is_silent):
        t = times[i]
        if not silent and not active:
            active = True; start = max(0, t - pad_before)
        elif silent and active:
            end = min(times[-1], t + pad_after)
            segments.append((start, end)); active = False
    if active:
        segments.append((start, times[-1]))

    # Объединение близких сегментов
    merged = []
    for s, e in segments:
        if merged and s - merged[-1][1] <= max_keep:
            merged[-1] = (merged[-1][0], max(merged[-1][1], e))
        else:
            if merged and s - merged[-1][1] > max_keep:
                merged.append((merged[-1][1], merged[-1][1] + max_keep))
            merged.append((s, e))
    return merged

def cut_silence(input_path, output_path, threshold=-40, min_silence=0.5,
                smooth=0.05, pad_before=0.1, pad_after=0.1, max_keep=0.5):
    video = VideoFileClip(input_path)
    audio_arr, sr = librosa.load(input_path, sr=None, mono=True)
    segments = detect_silence(audio_arr, sr, threshold, min_silence, smooth, pad_before, pad_after, max_keep)
    clips = [video.subclip(s, e) for s, e in segments if e - s > 0.05]
    if not clips:
        print("Ошибка: не найдено ни одного активного сегмента!")
        return
    final = concatenate_videoclips(clips)
    final.write_videofile(output_path, codec="libx264", audio_codec="aac")
    removed = video.duration - final.duration
    print(f"Готово! Удалено {removed:.1f}с из {video.duration:.1f}с ({removed/video.duration*100:.1f}%)")

if __name__ == "__main__":
    p = argparse.ArgumentParser()
    p.add_argument("input"); p.add_argument("output")
    p.add_argument("--threshold", type=float, default=-40)
    p.add_argument("--min-silence", type=float, default=0.5)
    p.add_argument("--pad", type=float, default=0.1)
    p.add_argument("--max-keep", type=float, default=0.5)
    args = p.parse_args()
    cut_silence(args.input, args.output, args.threshold, args.min_silence,
                pad_before=args.pad, pad_after=args.pad, max_keep=args.max_keep)
